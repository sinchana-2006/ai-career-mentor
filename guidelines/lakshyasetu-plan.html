Mobile App (2-hour plan)

Fastest path from React → native

1

Install Capacitor ~10 min

npm i @capacitor/core @capacitor/cli npx cap init LakshyaSetu com.lakshyasetu.app npx cap add android
2

Build & Sync ~5 min

npm run build npx cap sync
3

Open in Android Studio ~90 min

npx cap open android

Run on emulator or real device. Zero React code changes needed.
